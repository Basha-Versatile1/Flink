FLINK — SERVICE ILLUSTRATIONS
=============================

Drop one original illustration per service into this folder, then set the
`image` field on that service in src/data/services.ts (or ask Claude to wire
them up). Until an image is set, the site shows the coloured line icon instead.

FILENAMES (must match the service slug; .webp preferred, .png ok)
-----------------------------------------------------------------
  bathroom-cleaning.webp      → Bathroom Cleaning
  bedroom-cleaning.webp       → Bedroom Cleaning
  kitchen-cleaning.webp       → Kitchen Cleaning
  utensil-cleaning.webp       → Utensil / Dishwashing
  vacuuming.webp              → Vacuuming
  sweeping-mopping.webp       → Sweeping & Mopping
  whole-home-cleaning.webp    → Whole-Home Cleaning

SPECS
-----
  • Square, 512x512 (or 256x256), transparent OR very-light background
  • Consistent style + angle across all 7 so the grid looks cohesive
  • Keep file size small (aim < 40 KB each; export as WebP)

⚠️ ORIGINALITY (important)
--------------------------
Do NOT copy or lightly edit Pronto / Snabbit / any competitor's icons — that is
still infringement. Generate brand-new art in the OWN style below.

SUGGESTED ORIGINAL STYLE PROMPT (use with any AI image generator)
-----------------------------------------------------------------
Base style (keep identical for all 7):
  "Soft 3D clay-render icon, single object centred, isometric 3/4 angle,
   matte surfaces, gentle studio lighting and soft shadow, cyan (#06B6D4)
   accent details, clean light background, minimal, friendly, high detail,
   product-icon style."

Per-service subject (append to the base style):
  bathroom-cleaning   → "a sparkling bathroom sink with tap and a spray bottle"
  bedroom-cleaning    → "a neatly made bed with a folded blanket and pillow"
  kitchen-cleaning    → "a clean kitchen countertop with a hob and a cloth"
  utensil-cleaning    → "a stack of clean plates with a bowl and spoon"
  vacuuming           → "a small upright vacuum cleaner on a rug"
  sweeping-mopping    → "a broom and a mop bucket with sparkle marks"
  whole-home-cleaning → "a small tidy house with a sparkle/shine badge"

This style (clay-render, cyan accents, isometric) is deliberately different from
competitors' looks, so the results are original to Flink.
